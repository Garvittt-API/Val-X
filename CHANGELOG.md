## [0.1.0] - 2026-07-15

### Added

#### Core Features
- Real-time match tracking during agent select and in-game
- Player statistics display (K/D, HS%, ACS, ADR, KAST)
- Live scoreboard with ally and enemy teams
- Rank and RR tracking with visual indicators
- Peak rank and progression history

#### Dashboard
- Main dashboard with game state overview
- Client connection status indicator
- Current match summary with player list
- Personal profile with rank stats

#### Player Analytics
- Player search by Riot ID
- Match history with detailed statistics
- Win/loss tracking with RR changes
- Recent performance trends

#### In-Game Overlay
- Transparent overlay window for live matches
- Minimal and compact design
- Ally and enemy team display

#### UI/UX
- Modern glassmorphism design system
- 3D perspective card transforms
- Neon glow effects and animations
- Dark theme optimized for gaming
- Responsive sidebar navigation
- Custom title bar with window controls

#### Technical
- Tauri v2 backend with Rust
- React 18 with TypeScript
- Zustand state management
- TailwindCSS styling
- Vite build system
- WebSocket presence tracking
- Lockfile authentication with VALORANT client
